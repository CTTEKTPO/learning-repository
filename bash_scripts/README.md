Управление процессами через скрипт
Скрипт использует утилиту ps, для вывода данных исходя из переданных параметров

Параметры:
list                    - Показать все мои процессы
top-cpu [N]             - Топ N по CPU (по умолчанию 5)
top-mem [N]             - Топ N по памяти
find <процесс>          - Найти процесс по имени
kill <процесс>          - Убить процесс (SIGTERM)
kill-force <процесс>    - Убить процесс (SIGKILL)
info <PID>              - Показать инфо о процессе (ps + дополнительно)
